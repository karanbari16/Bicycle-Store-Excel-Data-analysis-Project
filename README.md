# Bicycle-Store-Excel-Data-analysis-Project
Interactive Excel dashboard analyzing retail sales performance, customer demographics, product profitability, and regional trends using Pivot Tables, Pivot Charts, Slicers, and advanced Excel functions.

📌 Project Overview

This project focuses on analyzing retail sales data for a bicycle and accessories retailer. The objective is to uncover valuable business insights related to sales performance, customer behavior, product profitability, and regional trends using Microsoft Excel.

The dashboard enables stakeholders to make data-driven decisions by providing a comprehensive overview of key business metrics and performance indicators.

🎯 Business Problem

The management team wants to understand:

-Which products generate the highest revenue and profit.
-Which customer segments contribute most to sales.
-Which regions perform best.
-How sales and profitability change over time.
-Which products and markets offer future growth opportunities.

🧹 Data Cleaning

The following data preparation steps were performed:

-Removed duplicates
-Handled missing values
-Standardized formatting
-Cleaned text using TRIM and CLEAN functions
-Verified revenue, cost, and profit calculations
